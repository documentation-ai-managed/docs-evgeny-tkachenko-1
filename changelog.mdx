## Recent Updates

Stay informed about all changes to Evgeny Tkachenko. You can find detailed release notes, migration guides, and contributor credits below. Updates are listed in reverse chronological order, with the most recent first.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added support for real-time collaboration in documentation pages, allowing multiple users to edit simultaneously.
- Introduced customizable themes with your brand color `#3B82F6` for personalized documentation spaces.
- New search functionality with full-text indexing across all project docs.

## Improvements

- Enhanced performance for large documentation sites, reducing load times by up to 40%.
- Improved mobile responsiveness for better viewing on all devices.

## Bug Fixes

- Fixed issue with broken internal links in generated MDX pages.
- Resolved parsing errors in nested code blocks.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "bugfix"]}>

## New Features

- Integrated Mermaid diagram support using standard code blocks for flowcharts and sequences.
- Added starter kit templates for quick documentation setup.

## Bug Fixes

- Corrected JSX component parsing issues related to special character escaping.
- Fixed heading hierarchy enforcement starting from H2 levels.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature", "breaking"]}>

## Initial Release

- Core MDX documentation generation with YAML frontmatter and component support.
- Built-in components including Callout, Cards, Steps, and Update for rich pages.
- Full compliance with parsing rules to prevent site errors.

## Breaking Changes

- Removed support for H1 headings in body content; always start with H2.

</Update>
