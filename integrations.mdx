## Custom API Integrations

Build tailored integrations using the REST API at `https://api.example.com/v1/integrations`. Authenticate with your API key.

<CodeGroup tabs="JavaScript,Python,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/integrations', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      name: 'Custom Webhook',
      endpoint: 'https://your-webhook-url.com/webhook'
    })
  });
  ```
  ```python
  import requests

  headers = {
      'Authorization': f'Bearer {YOUR_API_KEY}',
      'Content-Type': 'application/json'
  }
  data = {
      'name': 'Custom Webhook',
      'endpoint': 'https://your-webhook-url.com/webhook'
  }
  response = requests.post('https://api.example.com/v1/integrations', json=data, headers=headers)
  ```
  ```bash
  curl -X POST https://api.example.com/v1/integrations \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "name": "Custom Webhook",
      "endpoint": "https://your-webhook-url.com/webhook"
    }'
  ```
</CodeGroup>

<ParamField path="name" param-type="string" required="true">
  Integration display name.
</ParamField>

<ParamField path="endpoint" param-type="string" required="true">
  Target URL for events.
</ParamField>

## Webhook Configurations

Set up webhooks to receive real-time events like doc updates or publishes.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to Settings > Integrations in your dashboard at `https://dashboard.example.com`.
    
    Click "New Webhook" and enter your endpoint URL.
  </Step>
  <Step title="Verify Endpoint" icon="check-circle">
    Test the webhook with a sample payload:
    
````json
{
  "event": "doc.updated",
  "data": {
    "id": "doc-123",
    "title": "Updated Guide"
  }
}
````
  </Step>
  <Step title="Enable Events" icon="settings">
    Select events: `doc.published`, `doc.commented`, `user.invited`.
    
    Save and monitor incoming requests.
  </Step>
</Steps>

<Callout kind="tip">
  Use HMAC signatures for security. Verify payloads with the `X-Signature` header.
</Callout>

## Embedding and Export Integrations

Embed documentation or export content seamlessly.

<Tabs>
  <Tab title="Iframe Embed" icon="code">
    Embed pages using:
    
````html
<iframe src="https://docs.example.com/your-page" width="100%" height="600px"></iframe>
````

    Customize with query params: `?theme=dark&hide-toc=true`.
  </Tab>
  <Tab title="Markdown Export" icon="download">
    Export docs via API:
    
````bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://api.example.com/v1/docs/export \
  -o docs.md
````
  </Tab>
</Tabs>

